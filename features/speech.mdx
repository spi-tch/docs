## Request

The `generate()` function can be used to generate speech. The following parameters are available:

### Parameters

- `text` (required) - The text you want to convert to speech
- `language` (required) - Language code: `en`, `ha`, `ig`, `yo`
- `voice` (required) - Voice ID to use for generation
- `model` (optional) - Model to use, defaults to `legacy`
- `format` (optional) - Output audio format, defaults to `wav`

### Audio Formats

The `format` parameter supports the following audio formats:
- `wav` (default) - Standard wave format
- `mp3` - MPEG Layer III audio
- `ogg_opus` - OGG container with Opus codec
- `webm_opus` - WebM container with Opus codec  
- `flac` - Free Lossless Audio Codec
- `pcm_s16le` - Raw PCM 16-bit little-endian
- `mulaw` - μ-law encoded audio
- `alaw` - A-law encoded audio

## Response

The response for speech generation is in bytes.

- The Content-Type is `audio/wav`
- The content is streamed back to the caller.
- The file type of the generated audio is `wav`.
  If you use the streaming interface (Python SDK), you can start to take action on the byte chunks, e.g. stream to file.

## Choosing a Voice

We currently have 40 characters spanning four languages (English, Hausa, Igbo, Yoruba). Browse the roster on the [Voices](/concepts/voices) page and match voices to your use case before generating audio.

## Language Support

The speech generation model supports the following languages:

- English: `en`
- Hausa: `ha`
- Igbo: `ig`
- Yoruba: `yo`

## Examples

<CodeGroup dropdown>

```python Python
import os
from spitch import Spitch

os.environ["SPITCH_API_KEY"] = "YOUR_API_KEY"
client = Spitch()

with open("new.mp3", "wb") as f:
    response = client.speech.generate(
        text="Bawo ni ololufe mi?",
        language="yo",
        voice="sade",
        format="mp3"
    )
    f.write(response.read())
```

```typescript TypeScript
import Spitch from 'spitch';
import fs from 'fs';

const client = new Spitch({ apiKey: process.env.SPITCH_API_KEY });

async function generateSpeech(): Promise<void> {
    const res = await client.speech.generate({
        text: "Báwo ní ṣe wà?",
        language: "yo" as const,
        voice: "femi" as const,
        format: "wav" as const
    });
    
    const blob = await res.blob();
    const buffer = Buffer.from(await blob.arrayBuffer());
    fs.writeFileSync("output.wav", buffer);
    console.log("Audio saved to output.wav");
}

generateSpeech();
```
