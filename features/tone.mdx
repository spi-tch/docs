Many African languages are tonal (i.e. there is a specific way to pronounce them properly). In writing, this is indicated by certain marks which can help the reader (human or AI) to properly pronounce such text.
We have built this endpoint with model support for African languages.

## Request

The `tone_mark()` function can be used to add tone marks to a sentence. Examples are provided below as a guide for you.

### Best Practices for Use
<Info>
  When tone marking a sentence, you should use the language code (e.g. `yo`, `ig`) and not the full text.
</Info>


## Response

The response for speech generation is in bytes.

- The Content-Type is `application/json`
- A `request_id` is returned for issue resolution with our support team.

Below is an example of a response from the API

```json
{
  "request_id": "'86095cea-77d5-45ba-a093-0f800ac2c7df'",
  "text": "Báwo ni olólùfẹ́ mi?"
}
```

## Language Support

The speech generation model supports the following languages:

- Yoruba: yo

Support for Igbo is coming soon.
More info on languages can be found on the [Languages](/concepts/languages) page



## Examples

<CodeGroup dropdown>

```python Python
import os
from spitch import Spitch

os.environ["SPITCH_API_KEY"] = "YOUR_API_KEY"
client = Spitch()
response = client.text.tone_mark(
    language="yo",
    text="Bawo ni ololufe mi?"
)
print(response.text)   # Báwo ni olólùfẹ́ mi?
```


```typescript TypeScript
import Spitch from 'spitch';

const client = new Spitch({ apiKey: process.env.SPITCH_API_KEY });

async function addToneMarks(): Promise<void> {
    const response = await client.text.toneMark({
        text: "Omo omo ni ayo",
        language: "yo" as const
    });
    console.log("With diacritics:", response.text);
}

addToneMarks();
```
