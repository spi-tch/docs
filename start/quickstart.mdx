    <Tab title={"Transcribe audio"} icon={"message-text"}>
        <Steps>
            <Step title="Get an API Key" icon="key">
                <Columns>
                    <Frame caption={"Creating a Spitch API key"}>
                        <img src={"/images/create-api-key.png"} alt={"Creating a Spitch API key"}/>
                    </Frame>
                    <Frame caption={"API key has been created"}>
                        <img src={"/images/api-key-created.png"} alt={"Screenshot of API key created"}/>
                    </Frame>
                </Columns>
                Create your API key at [Spitch Studio](https://spitch.studio/api-keys). This will be used to [securely access the API](/keys).
            </Step>

            <Step title="Install your library" icon="sign-posts-wrench">
                Our official Python and Node SDKs allow you easily use the API. You can also install `dotenv` so that your API keys are set properly.
                <CodeGroup dropdown>
                    ```python Install the official Spitch Python SDK
                    pip install spitch
                    pip install python-dotenv
                    ```

                    ```javascript Install the official Spitch Node SDK
                    npm install spitch
                    npm install dotenv
                    ```
                </CodeGroup>
                More info on installation can be found on [Installation](/getting-started/installation)
            </Step>
            <Step title={"Set environment variable"} icon={"envira"}>
                Set your environment variable depending on your OS
                <CodeGroup dropdown>
                    ```bash MacOS/Linux
                    export SPITCH_API_KEY="your_api_key_here"
                    ```

                    ```powershell Windows Powershell
                    setx SPITCH_API_KEY "your_api_key_here"
                    ```
                </CodeGroup>
                Alternatively, you can set it in your `.env` file
                ```.env
                SPITCH_API_KEY=<your_api_key>
                ```
            </Step>
            <Step title="Write your script" icon="square-terminal">
                <CodeGroup dropdown>
                    ```python app.py
                    from spitch import Spitch
                    client = Spitch(api_key="YOUR-API-KEY")

                    response = client.speech.transcribe(
                        content=open("file.wav", "rb"),
                        language="en",
                        model="mansa_v1"
                    )

                    print(response.text)
                    ```

                    ```javascript app.js
                    import { Spitch } from "spitch";
                    const client = new Spitch();

                    const response = await client.speech.transcribe({
                        language: 'en',
                        content: fs.createReadStream('file.wav')
                        model: "mansa_v1",
                    });

                    console.log(response.text);
                    ```
                </CodeGroup>
            </Step>

            <Step title="Run the code" icon="person-running-fast">
                <CodeGroup dropdown>
                    ```python
                    python app.py
                    ```

                    ```javascript
                    npx node app.js
                    ```
                </CodeGroup>
                You should see the text displayed on console.
            </Step>
        </Steps>
    </Tab>

                    ```python app.py
                    from spitch import Spitch
                    client = Spitch(api_key="YOUR-API-KEY")

                    with open("audio.mp3", "wb") as f:
                        response = client.speech.generate(
                            text="I'll be in the backyard of the Ikorodu house.",
                            language="en",
                            voice="lina"
                        )
                        f.write(response.read())                    
                    ```

                    ```javascript app.js
                    import Spitch from 'spitch';
                    import fs from 'fs';

                    const client = new Spitch({ apiKey: process.env.SPITCH_API_KEY });

                    async function generateSpeech() {
                    const res = await client.speech.generate({
                        text: "I'll be in the backyard of the Ikorodu house.,
                        language: "en",
                        voice: "lina"
                    });
                    const blob = await res.blob();
                    const buffer = Buffer.from(await blob.arrayBuffer());
                    fs.writeFileSync("output.wav", buffer);
                    console.log("Audio saved to output.wav");
                    }

                    generateSpeech();
                    ```
