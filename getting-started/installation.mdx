You can use our Python SDK to interact with the models
<Note>
    If Python isn't your vibe, try using the ReST API directly. Go to the API Reference section.
    We're working to add SDKs for more languages.
</Note>

## Pre-requisites
To install the Spitch Python library, you need to have the following:
* Python >=3.7

## Install
Run the following command to install the Spitch Python SDK.
```python
pip install spitch
```
